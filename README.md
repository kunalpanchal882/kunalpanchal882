# Resume Screening Application

A modern full-stack web application focused on intelligent resume analysis and efficient candidate screening.

Built with **React**, **FastAPI**, and **Google Gemini AI**, the platform combines clean user experience with powerful backend processing to deliver fast, accurate, and scalable performance.

Designed with a strong focus on usability, maintainable architecture, responsive design, and seamless API integration.

### Tech Stack

**Frontend:** React, Vite, Tailwind CSS  
**Backend:** FastAPI, Python  
**AI Integration:** Google Gemini API  

---

A practical project showcasing full-stack development, API architecture, AI integration, and modern web engineering practices.
